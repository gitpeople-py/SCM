# SCM
🛡️ SCM — Защита от читеров в Minecraft
SCM Scanner — это мощный инструмент для администраторов Minecraft-серверов, позволяющий обнаруживать читерские программы, моды и инжекторы на компьютере игрока. Программа сканирует папку .minecraft и активный процесс Minecraft, находя подозрительные файлы и компоненты.


⚡ Основные функции:

Сканирование модов
Просмотр всех установленных модов

Автоматическое обнаружение чит-клиентов (Wurst, Impact, Meteor, LiquidBounce и др.)

Подсветка подозрительных модов красным цветом

Сканирование DLL файлов
Поиск DLL файлов в папке Minecraft

Обнаружение подозрительных библиотек (инжекторы, байпасы)

Обнаружение инжектированных DLL
Проверка активного процесса Minecraft (javaw.exe)

Обнаружение чит-инжекторов в реальном времени

Работает только при запущенной игре

Поиск подозрительных папок
Сканирование папок в .minecraft на наличие следов читов

Обнаружение папок с названиями: wurst, impact, vape, meteor, config, alts, proxy и др.

📄 Экспорт отчёта
Сохранение полного отчёта в формате JSON

Автоматическое сохранение в C:\SCM\

Подпись датой и временем для удобства


🔐 Безопасность и конфиденциальность
Программа работает локально — данные не отправляются никуда

Отчёты можно передавать администратору сервера для проверки

Пользователь сам решает, кому показывать отчёт

🖥️ Системные требования
Компонент	Требование
ОС	Windows 7/8/10/11
Python	3.6 или выше (для запуска из кода)
Память	от 100 МБ
Диск	от 50 МБ
Дополнительно	Запуск от имени администратора для сканирования инжектированных DLL
📁 Экспорт отчёта
При нажатии кнопки "📄 Экспорт отчёта" создаётся файл:


C:\SCM\SCM_Report_20250618_143025.json
Файл содержит:

Информацию о времени сканирования

Путь к папке Minecraft

IP и порт сервера (если игра запущена)

Полный список модов, DLL, папок

Помеченные подозрительные файлы

Статистику сканирования

📖 Пример использования
Администратор просит игрока запустить SCM 

Игрок нажимает "🔄 Обновить всё"

Нажимает "📄 Экспорт отчёта"

Отправляет файл из C:\SCM\ администратору

Администратор открывает файл в SCM Report Reader или текстовом редакторе

Принимает решение о выдаче бана или предупреждения

⚠️ Важно!
SCM НЕ является автоматическим античитом — это инструмент для ручной проверки

Программа не блокирует и не удаляет файлы — только показывает информацию

Для обнаружения инжектированных DLL требуется запуск от имени администратора

Некоторые читы могут не оставлять следов в папке .minecraft (внешние клики, Cheat Engine)


Исходный код открыт для модификации

База подозрительных слов регулярно обновляется
